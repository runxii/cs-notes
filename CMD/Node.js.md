Node.js can be installed by [[nvm]]

## Install
``` bash
nvm ls-remote // check the lastest available version
nvm install v22.11.0 // install the version
```

## Set default version
``` bash
nvm alias default 16.15.0
```

## Removing Node.js
``` bash
nvm uninstall <node_version>
```
